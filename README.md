# 📈 Scanner de Marché Prédictif - IA

Ce projet utilise l'Intelligence Artificielle (**Random Forest**) pour analyser et prédire la direction des actifs financiers.

## 🚀 Fonctionnalités
- **Analyse Automatisée** : Récupération des données en temps réel via `yfinance`.
- **Indicateurs Techniques** : Calcul du RSI, SMA-5, SMA-20 et de la Volatilité.
- **Machine Learning** : Modèle de classification pour détecter les signaux d'achat/vente.
- **Reporting** : Export automatique des prédictions vers un fichier Excel horodaté.

## 🛠️ Technologies utilisées
- **Langage** : Python 3
- **Librairies** : Pandas, Scikit-Learn, yFinance, OpenPyXL
- **Interface** : Jupyter Notebook & Script autonome (.py)
